# Módulo 01 — Embeddings e Bancos de Dados Vetoriais

## O que foi desenvolvido

Implementação prática de busca semântica aplicada a projetos
de engenharia elétrica usando SentenceTransformers e FAISS.

## Tecnologias
- Python 3.13
- SentenceTransformers 5.3.0
- FAISS-CPU
- PyTorch 2.10
- Hugging Face Hub

## Notebooks
- `pos.ipynb` — embeddings, similaridade semântica e busca vetorial

## Resultados obtidos
- Similaridade semântica: 0.4611 entre projetos relacionados
- Similaridade semântica: -0.0742 entre temas não relacionados  
- 8 projetos indexados e buscados em 0.2 segundos
